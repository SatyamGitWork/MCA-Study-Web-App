1. Emmet Documentation & Example

Emmet is a plugin for many popular text editors that significantly speeds up HTML and CSS development. It uses abbreviations (snippets) that are expanded into full, valid code when you press a key (usually Tab).

Emmet Abbreviation:

ul.nav>li.item*3>a{Item $}

Expanded HTML:

<ul class="nav">
    <li class="item"><a href="">Item 1</a></li>
    <li class="item"><a href="">Item 2</a></li>
    <li class="item"><a href="">Item 3</a></li>
</ul>

2. Flexbox

Flexbox (Flexible Box Layout) is a one-dimensional CSS layout model for arranging items in rows or columns. It makes it easy to align items, distribute space, and reorder content.

Code:

<!-- HTML -->
<div class="flex-container">
    <div class="item">Item 1</div>
    <div class="item">Item 2</div>
    <div class="item">Item 3</div>
</div>

<!-- CSS -->
<style>
.flex-container {
    display: flex;
    justify-content: center;
    align-items: center;
    background-color: #f0f0f0;
    height: 200px;
    border: 1px solid #ccc;
}
.item {
    background-color: #007bff;
    color: white;
    padding: 20px;
    margin: 10px;
    border-radius: 5px;
}
</style>

3. Table Tag

The <table> tag is used to display data in a grid of rows and columns.

  • <table>: The wrapper for the entire table.
  • <tr>: (Table Row) Defines a row.
  • <th>: (Table Header) Defines a header cell. Text is bold and centered by default.
  • <td>: (Table Data) Defines a data cell.

Code:

<table border="1">
  <thead>
    <tr>
      <th>First Name</th>
      <th>Last Name</th>
      <th>Age</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John</td>
      <td>Doe</td>
      <td>30</td>
    </tr>
    <tr>
      <td>Jane</td>
      <td>Smith</td>
      <td>25</td>
    </tr>
  </tbody>
</table>

4. Different Types of CSS

There are three ways to add CSS to an HTML document:

  1. Inline CSS: Using the style attribute directly on an HTML element.
  2. Internal CSS: Using a <style> tag within the <head> section of the HTML.
  3. External CSS: Linking to a separate .css file using a <link> tag in the <head>.

Code:

<!DOCTYPE html>
<html>
<head>
    <title>Types of CSS</title>

    <!-- 3. External CSS (links to styles.css) -->
    <link rel="stylesheet" href="styles.css">
    <!-- styles.css contains:
         body { background-color: #f4f4f4; } 
    -->

    <!-- 2. Internal CSS -->
    <style>
        h2 {
            color: green;
        }
    </style>
</head>
<body>
    <!-- 1. Inline CSS -->
    <p style="color: blue; font-size: 18px;">
        This uses Inline CSS.
    </p>

    <h2>This uses Internal CSS</h2>

    <p>Page background is from External CSS.</p>
</body>
</html>

5. Differentiate: Block vs. Inline

Feature Block-level Element Inline-level Element
New Line Always starts on a new line. Stays on the same line (in the "flow" of text).
Width Takes up the full width available by default. Takes up only as much width as its content needs.
Height/Width You can explicitly set width and height. width and height properties have no effect.
Margin/Padding All margin and padding are respected. Only horizontal margin/padding are respected. Vertical ones do not affect layout.
Examples <div>, <p>, <h1>, <ul> <span>, <a>, <strong>, <img>

6. Differentiate: Margin vs. Padding

Both create space, but in different places relative to the element's border.

Property Margin Padding
Location Space OUTSIDE an element's border. Space INSIDE an element's border (between content and border).
Purpose Creates space between elements (pushes other elements away). Creates space within an element (gives content "breathing room").
Background Always transparent. Shows the parent's background. Shows the background color/image of the element itself.
Collapse Vertical margins can "collapse" (smaller margin is ignored). Padding never collapses.

7. Differentiate: Container vs. Empty Tag

Feature Container Tag Empty Tag (Void Element)
Structure Has both an opening tag (<tag>) and a closing tag (</tag>). Is a single, self-closing tag (e.g., <tag> or <tag />).
Content Designed to hold content (text or other tags) between the tags. Cannot hold any content.
Purpose Defines a block or section of content. Inserts a specific item (line break, image, horizontal rule).
Examples <div>...</div>, <p>...</p> <br>, <img>, <hr>, <input>

8. Differentiate: <p> vs. <pre>

Feature <p> (Paragraph) <pre> (Preformatted)
Whitespace Collapses whitespace. Multiple spaces, tabs, and newlines are treated as a single space. Preserves whitespace. Renders all spaces, tabs, and newlines exactly as written.
Font Renders in the page's default proportional font (like Arial). Renders in a monospace font (like Courier) by default.
Use Case Standard paragraphs of text. Displaying code snippets, poetry, or text where spacing is vital.

9. Differentiate: <bdo> vs. <dir>

Tag <bdo> (Bi-Directional Override) <dir> (Directory)
Purpose Explicitly overrides the text direction (e.g., for right-to-left languages). DEPRECATED (Obsolete). Was for a directory listing.
Status A valid HTML tag for internationalization (i18n). Do not use. Removed from HTML5.
Usage This is <bdo dir="rtl">reversed</bdo>. (N/A - Do not use)
Alternative (N/A) Use <ul> (unordered list) for all lists.

10. Fieldset & Legend

The <fieldset> tag groups related elements in a form, and <legend> provides a caption for that group. This is great for accessibility and organization.

Code:

<form>
  <fieldset>
    <legend>Personal Information</legend>
    <label for="fname">First Name:</label>
    <input type="text" id="fname" name="fname">
    <br><br>
    <label for="lname">Last Name:</label>
    <input type="text" id="lname" name="lname">
  </fieldset>
  <br>
  <fieldset>
    <legend>Preferences</legend>
    <input type="checkbox" id="news" name="news">
    <label for="news">Subscribe</label>
  </fieldset>
</form>

11. CSS Selectors

This example shows the most common CSS selectors in action.

Code:

<!DOCTYPE html>
<html>
<head>
    <title>CSS Selectors</title>
    <style>
        /* 1. Element Selector (all <p>) */
        p { font-family: Arial; }

        /* 2. Class Selector (.highlight) */
        .highlight { background-color: yellow; }

        /* 3. ID Selector (#main-title) */
        #main-title { color: #007bff; }

        /* 4. Attribute Selector */
        input[type="text"] { border: 2px solid blue; }

        /* 5. Descendant Combinator */
        ul li { color: green; }
    </style>
</head>
<body>
    <h1 id="main-title">Styled by ID</h1>
    <p>Styled by Element selector.</p>
    <p class="highlight">Styled by Class selector.</p>
    <ul>
        <li>Styled by Descendant.</li>
    </ul>
    <input type="text" placeholder="Styled by Attribute">
</body>
</html>
