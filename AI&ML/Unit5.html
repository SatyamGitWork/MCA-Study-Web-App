Machine Learning (ML)

A subset of AI that focuses on developing systems capable of learning and improving automatically from data without explicit programming.

Automation Data-Driven Adaptability Personalization

Hierarchy & Differences

AI (Broadest) Creating smart machines mimicking human behavior. (e.g., Robotics, Expert Systems)
ML (Subset of AI) Systems learning from data to make predictions. Moderate data needed. (e.g., Spam Filter)
DL (Subset of ML) Multi-layered neural networks. High complexity, needs huge data & GPUs. (e.g., Self-driving cars)

Supervised

Learns from Labeled Data (Input + Known Output).

  • Regression: Continuous output (e.g., Price).
  • Classification: Categories (e.g., Spam/Not Spam).
  • Algos: Linear Regression, SVM, Decision Trees.

Unsupervised

Learns from Unlabeled Data (Finding hidden structures).

  • Clustering: Grouping (e.g., Customer Segments).
  • Dimensionality Reduction: PCA.
  • Algos: K-Means, Hierarchical, PCA.

Reinforcement

Learning via Interaction (Agent, Environment, Reward).

  • Goal: Maximize cumulative reward.
  • Policy: Strategy for action.
  • Apps: Gaming (AlphaGo), Robotics.
Features (X) Independent variables/inputs (e.g., Age, Income).
Labels (Y) Dependent variable/outcome (e.g., Has Diabetes).
Data Splitting
Training Data (70-80%)
Teaches the model Testing (20%)
Evaluates

Overfitting (High Variance)

Good on Training, Bad on Testing.

Fix: More data, Simplify model, Regularization.

Underfitting (High Bias)

Bad on Training, Bad on Testing.

Fix: Increase complexity, Add features.

Evaluation Metrics

Metric Formula Use Case
Accuracy (TP+TN)/Total Balanced data
Precision TP/(TP+FP) Minimize False Positives (Spam)
Recall TP/(TP+FN) Minimize False Negatives (Cancer)
F1 Score 2*(P*R)/(P+R) Imbalanced data
